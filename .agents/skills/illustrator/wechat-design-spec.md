```css

    --art-soft-pink: #fce4ec;
    --art-lavender: #f3e5f5;

    --art-line: #5d4e37;
    --art-line-light: #6b5b45;

```

**色板设计理念**：
- 主色 `#7c6ba0`：优雅紫，提升视觉存在感
- 辅助色 `#e8a4b8`：柔和粉，形成鲜明层次对比
- 强调色 `#e89b5a`：暖橙，与暖色调背景和谐融合
- 线条色 `#6b5b45`：深棕，确保 WCAG AA 对比度标准

### 1.7 色彩应用原则

**60-30-10 法则**：
- 主色 60%：背景、大面积色块
- 辅助色 30%：次要元素、边框
- 强调色 10%：重点突出

**对比度标准**（WCAG AA 严格遵循）：

| 元素类型 | 最小对比度 | 推荐对比度 | 示例 |
|----------|-----------|-----------|------|
| 正文文字 | 4.5:1 | 7:1 | `#5d4e37` on `#fef9f3` = 7.2:1 ✓ |
| 大标题（≥18px） | 3:1 | 4.5:1 | `#6b5b45` on `#fef9f3` = 5.8:1 ✓ |
| 副标题/说明 | 4.5:1 | 6:1 | `#6b5b45` on `#fef9f3` = 5.8:1 ✓ |
| 图形元素 | 3:1 | 4:1 | 边框、图标等 |

**对比度验证工具**：
- 在线工具：[WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)
- 设计软件：Figma/Axure 内置对比度检测

## 三、布局原则

### 3.1 视觉平衡

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| 对称平衡 | 左右/上下对称 | 正式、庄重 |
| 不对称平衡 | 大小、颜色、位置调整 | 现代、活力 |
| 径向平衡 | 围绕中心辐射 | 聚焦、引导 |

### 3.2 留白原则

| 设计类型 | 留白比例 | 效果 |
|---------|---------|------|
| 高端设计 | 40-60% | 奢华、优雅 |
| 标准设计 | 30-40% | 平衡、舒适 |
| 信息密集 | 20-30% | 紧凑、高效 |

### 3.3 间距系统

基础单位：8px

| 间距 | 值 | 用途 |
|------|-----|------|
| xs | 8px | 紧凑间距 |
| sm | 16px | 元素内间距 |
| md | 24px | 元素间距 |
| lg | 32px | 区块间距 |
| xl | 40px | 大区块间距 |
| xxl | 48px | 章节间距 |

### 3.4 有机布局技巧

避免规则化、网格化的机械布局：

```css
.organic-item:nth-child(1) {
    transform: rotate(-2deg) translateY(10px);
}

.organic-item:nth-child(2) {
    transform: rotate(1deg) translateY(-5px);
}
```

**推荐做法**：
- 不规则的元素排列
- 有机的曲线和形状
- 变化的间距和尺寸
- 柔和的过渡和融合

## 四、微信平台规范

### 4.1 图片规格

| 图片类型 | 尺寸 | 宽高比 | 文件大小 |
|----------|------|--------|----------|
| 封面大图 | 900×383 | 2.35:1 | ≤2MB |
| 封面次图 | 900×500 | 1.8:1 | ≤2MB |
| 封面小图 | 200×200 | 1:1 | ≤500KB |
| 文中插图 | 900×自适应 | - | ≤1MB |
| 头图 | 900×300 | 3:1 | ≤500KB |
| 尾图 | 900×200 | 4.5:1 | ≤300KB |

### 4.2 安全区域

**封面图安全区域**：
- 顶部：60px
- 底部：40px
- 左右：40px

### 4.3 移动端适配

- 设计基准：iPhone 14 Pro Max (430×932)
- 最小可读字号：14px
- 点击区域最小：44×44px

## 六、设计检查清单

- [ ] 色彩搭配和谐（60-30-10法则）

- [ ] 分辨率符合要求
